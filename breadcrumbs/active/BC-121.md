## Problem

The compliance report shows CAA as "Not collected" because CAA lookup is an on-demand endpoint, not a scan field. The CAA metric is always N/A in the compliance report.

## Suggested fix

Store CAA lookup results per scan (present/absent, record value) so the compliance report can show a real percentage. Planned for 1.1.
