# BC-045: OpenAPI / Swagger UI not exposed for API endpoints

**Type:** todo
**Severity:** low
**Status:** new
**Filed:** 2026-06-01

## Problem

FastAPI generates OpenAPI schemas automatically, but there is no Swagger UI or ReDoc endpoint exposed. The README lists API endpoints in a markdown table, which is manual and prone to staleness. API consumers would benefit from an interactive documentation page.

## Suggested fix

Expose `/docs` and `/redoc` routes. Since all `/api/*` routes are auth-gated, the docs page should either be public (with a note that endpoints require auth) or gated behind the same auth middleware. This is a nice-to-have, not urgent.
