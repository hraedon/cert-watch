## Problem

The certificate detail page does not have an on-demand "Check revocation" button that triggers OCSP/CRL checks interactively, as shown in the 2026-06-02 design.

## Suggested fix

Add a button on the certificate detail page that calls the revocation-check endpoint and displays the results inline.
