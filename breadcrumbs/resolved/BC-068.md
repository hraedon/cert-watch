# BC-068: Security hardening batch (2026-06-01 round 2)

**Type:** security
**Severity:** medium
**Status:** resolved (2026-06-01)
**Filed:** 2026-06-01

## Problems fixed

1. **_probe_hsts SSRF bypass**: HSTS probe re-resolved hostname independently
   of the pinned IP. Now accepts and uses `pinned_ip` parameter (BC-062).

2. **DNS parser infinite loop**: Added cycle detection to `_parse_dns_name`
   and TXID/RCODE validation to `_resolve_with_dns` (BC-065).

3. **LDAP cleartext credentials warning**: When `ldap://` is used without
   STARTTLS and bind credentials are configured, emits WARNING at init time.

4. **OAuth state bypass**: `if state:` → `if not state:` (BC-067 #1).

5. **Open redirect via Host header**: Added `CERT_WATCH_BASE_URL` env var
   for OAuth redirect URI construction (BC-067 #2).

6. **Silent auth degradation**: `build_auth_provider` raises ValueError on
   misconfiguration instead of falling back to NoAuthProvider (BC-066).

7. **Scrypt weak parameter warning**: Rejects hashes with `n<2` or `r<1`
   or `p<1`; warns on `n<16384` or `r<8`.

8. **Rate limiting proxy bypass**: views.py and auth.py login rate limit
   now use `_extract_client_ip()` instead of `request.client.host`.

9. **DOM XSS in pivot CSS class**: `escHtml()` alone doesn't prevent CSS
   class injection; added `_VALID_URGENCIES` whitelist for urgency values.

10. **CSP and security headers**: Added `security_headers_middleware` with
    Content-Security-Policy, X-Content-Type-Options, X-Frame-Options.

11. **CSV injection**: `_csv_safe()` escapes cells starting with `=`, `+`,
    `-`, `@`, `\t` in CSV exports.

12. **Healthz leak**: Exception details in healthz now return generic
    "error" instead of raw exception text.

13. **Cookie path**: All session/CSRF/OAuth cookies now include `path="/"`.

14. **Break-glass detection**: Uses `isinstance()` instead of `type().__name__`.

15. **SQL injection surface documented**: `_load_unified_filtered()` uses
    f-string SQL for `host_where`; documented as internal-only / whitelisted.
