## Problem

The webhook alert configuration required manual JSON template construction. Presets for common targets (Alertmanager, Slack, PagerDuty, Teams) would improve UX.

## Resolution

Added webhook alert presets in the settings UI:
- `settings.html` now includes preset buttons for Slack, Teams, PagerDuty, and Alertmanager
- Each preset pre-fills the `webhook_url` and `webhook_template` with the target-specific format
- The `alert_adapters.py` registry already supported these targets (Plan 022)
- `ALERT_WEBHOOK_KIND` env var selects the adapter at runtime
