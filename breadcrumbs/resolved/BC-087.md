severity: medium

**Found 2026-06-02 code audit.** The `POST /api/webhook/test` endpoint in
`routes/api.py` used `Depends(require_auth)` instead of `Depends(require_write)`.
This allowed read-only users (BC-086) to trigger webhook sends — a side-effecting
mutation that should be gated by the write check.

**Fix:** Changed to `Depends(require_write)`. All 694 tests pass.
