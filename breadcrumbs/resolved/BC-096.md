---
identifier: BC-096
title: Decouple test monkeypatching from route-level import names
kind: improvement
status: new
severity: low
---

## Problem

Many tests monkeypatch route-level import names (e.g., `monkeypatch.setattr("cert_watch.alerts.ssrf_safe_urlopen", ...)`). This is fragile because the import names can change.

## Suggested fix

Refactor tests to monkeypatch at the source module rather than the route-level import, or use dependency injection.
