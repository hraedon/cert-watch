## Problem

`test_bc083_081.py` and `test_middleware_deps.py` call `asyncio.run()` inside pytest-asyncio tests that already run inside a live event loop. This raises:

`RuntimeError: asyncio.run() cannot be called from a running event loop`

The `_drive_lifespan` helper uses `asyncio.run()` to enter the app lifespan.

## Fix

Either use `asyncio.get_event_loop().run_until_complete()` or convert the test functions to async-native and await directly. This is a fundamental incompatibility with the pytest-asyncio runner, not a regression in the app.
