**Resolved:** 2026-05-29

## Resolution

Decomposition complete. `app.py` reduced from ~1500 lines to ~121 lines. Route handlers extracted into `routes/` package (api.py, auth.py, certificates.py, hosts.py, views.py). Middleware moved to `middleware.py`, template filters to `filters.py`. `app.py` now contains only the app factory, lifespan, static file mount, and middleware registration.
