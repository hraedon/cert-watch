## Problem

Dashboard sorting and notes required full page reloads. Adding HTMX or fetch-based progressive enhancement would improve UX.

## Resolution

Implemented vanilla JS progressive enhancement for host-level notes on the dashboard:
- Added inline note editing via `fetch()` to `POST /api/hosts/{id}/notes`
- Notes are saved without page reload; success/error is shown inline
- The `dashboard.html` template includes a `{% block scripts %}` with the fetch handler
- The existing `test_host_notes.py` covers the API endpoint
