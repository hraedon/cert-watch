## On the work done

The core module (`compliance.py`) came together cleanly — pure data aggregation with no I/O in the model layer. The signing/verification is a straightforward HMAC-SHA256 on canonical JSON, which matches what the plan specified.

The biggest time sink was the test seeding helper (`_seed_fleet`). Raw SQL with positional placeholders is fragile — I got the column/placeholder count wrong three times before getting it right. The lesson: use the repository classes or upload helpers, not hand-crafted INSERT statements. I filed BC-123 for this.

The HTML template was straightforward — print CSS via `@media print`, grade chips, metric bars. The inline style budget test caught the new template and the updated insights page immediately, which is exactly what it's supposed to do.
